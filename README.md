# Getting Started
Run `make` to run all of the following setup scripts.
# Custom Installations
## Homebrew Setup
Simply download and run `sh brew.sh`.
## Homebrew Casks Setup
Simply download and run `sh casks.sh`.
## macOS Setup
Simply download and run `sh macOS.sh`.
